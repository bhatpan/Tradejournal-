☁ Supabase Database

Connect to a Supabase project to sync your journal across devices. Your data is stored in your own database.

Setup guide
1. Create a free project at supabase.com
2. Go to SQL Editor and run:
CREATE TABLE tradeedge_data (
  key TEXT PRIMARY KEY,
  data JSONB,
  updated_at TIMESTAMPTZ DEFAULT now()
);
ALTER TABLE tradeedge_data ENABLE ROW LEVEL SECURITY;
CREATE POLICY "public_access" ON tradeedge_data FOR ALL USING (true) WITH CHECK (true);
3. Go to Project Settings → API and copy the URL and anon key above
4. Click Save & Connect, then Push Local → Supabase
